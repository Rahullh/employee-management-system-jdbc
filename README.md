# Employee Management System (JDBC)

A desktop-based Employee Management System developed using Java Swing and JDBC to manage employee records.

## Features
- Add employee details
- View employee records in a JTable
- Update employee information
- Delete employee records

## Tech Stack
- Java
- JDBC
- Swing
- MySQL
- Eclipse

## Project Structure
src/
 ├── db/
 ├── dboperations/
 ├── model/
 ├── ui/
 └── main/

## How to Run
1. Create a MySQL database named `ems_db`
2. Create an `employee` table
3. Update MySQL username and password in `DBConnection.java`
4. Run `MainApp.java` from Eclipse
